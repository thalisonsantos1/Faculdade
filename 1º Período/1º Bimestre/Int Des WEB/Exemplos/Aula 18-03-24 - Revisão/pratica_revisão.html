Seção 02 - Exemplo de Estrutura HTML

Os documentos HTML possuem uma estrutura básica, a qual geralmente compõe a maioria das páginas HTML. Veja o exemplo abaixo: 

                
                <!DOCTYPE html>

                <html lang="pt-br">

                    <head>

                        <title> Aqui vai o título </title>

                        <meta charset="utf-8">

                    </head>

                    <body>

                    </body>

                </html>

Aqui pode ser inserido um pequeno texto explicando a estrutura de forma básica.
